# Julie Countdown Timer

簡單的倒數計時器網頁，支援天、時、分、秒輸入，並提供開始、暫停、重設功能。

## 檔案

- `countdown-demo.html`: 主要倒數計時器頁面
- `index.html`: GitHub Pages 入口頁，會自動跳轉到計時器

## 使用方式

1. 開啟 `countdown-demo.html` 或 `index.html`
2. 輸入天、時、分、秒
3. 按下「開始」進行倒數

## GitHub Pages

如果你要直接用 GitHub 網址開啟這個頁面，可以在 GitHub 倉庫的 Pages 設定中選擇：

- Branch: `main`
- Folder: `/ (root)`

啟用後通常可從這個網址開啟：
